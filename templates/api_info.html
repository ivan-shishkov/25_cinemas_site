Что в кино API

Данный сервис предоставляет публичное API, позволяющее получить информацию о топ-10 фильмах с самым высоким рейтингом, идущих сегодня в кинотеатрах Москвы, в формате JSON. Для этого необходимо обратиться по адресу /api/top-rated-movies

Формат ответа:

          {
              "top_rated_movies_info": [
                  {
                      "afisha_url": ...,
                      "age_restriction": ...,
                      "country": ...,
                      "description": ...,
                      "duration": ...,
                      "name": ...,
                      "rating": ...,
                      "screenshot_url": ...,
                      "votes": ...,
                      "year": ...
                  },
                  ...
              ]
          }
  • afisha_url - ссылка на Афиша.ру с информацией о фильме
  • age_restriction - возрастные ограничения
  • country - страна производства
  • description - описание фильма
  • duration - продолжительность фильма
  • name - название фильма
  • rating - рейтинг фильма на Кинопоиск.ру
  • screenshot_url - ссылка на скриншот из фильма
  • votes - количество голосов на Кинопоиск.ру
  • year - год производства

Настоящий сайт создан в исключительно учебных целях. Все сведения о фильмах взяты с сайтов Афиша и Кинопоиск.
